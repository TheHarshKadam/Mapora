# 🌍 Mapora

> **Explore the world one guess at a time.**

Mapora is an interactive geography game where players guess a hidden country based on distance feedback. The closer your guess, the hotter you get 🔥

---

## ✨ Features

- 🌐 Interactive 3D globe experience  
- 📍 Distance-based hint system (Hot/Cold feedback)  
- 🎯 Smart guess tracking & ranking  
- 🎨 Beautiful dark/light UI  
- ⚡ Fast & responsive gameplay  
- 🔜 Multiplayer mode (coming soon)

## 🚀 How It Works

1. Start the game  
2. Enter a country guess  
3. Get distance feedback  
4. Use clues to refine your guesses  
5. Find the hidden country in minimum attempts  

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Three.js / Three Globe
- Tailwind CSS

**Backend**
- Node.js
- Express.js

**Other**
- REST APIs  
- Country dataset (GeoJSON)

## 📦 Installation

```bash
# Clone the repo
git clone https://github.com/your-username/mapora.git

# Navigate into project
cd mapora

# Install dependencies
npm install

# Run the app
npm run dev
```
